***TITLE
Array explorer
***SCREENSHOT
images/ArrayChooser.png
***PROLOGUE
This component allows an analyst to compare the rows and columns of an array of data.
***CODE
element.array_explorer.example(element);
***DISCUSSION
This widget allows the user to interactively example a data array.
<p>
Click an entry to change the currently selected row and column.  Use the colorize button to fix
multiple row/column pairs for comparison.  Click and drag to select a subsection of the array.
