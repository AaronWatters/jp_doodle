Image detail viewer.

This demo shows a large image in four panels. The left images show the whole image at low resolution and the right panel shows part of the image in full resolution.

Drag the mouse over the left panel to change the region of interest viewed in the right panel. Use the dropdowns to adjust the zoom level. The detail-drag/zoom events are synchronizedÂ between the top and bottom panels.

Adjust the sliders to change the "gamma" contrast for the images. The contrast for the top panels the bottom panels are independent.

Please consult the repository source code for canvas_image_detail usage.
